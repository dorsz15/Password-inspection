# 🛡️ Password Fortress & Analyzer

A modern web application built with React, TypeScript, and Vite to generate secure pseudorandom passwords and analyze their cryptographic strength using the `zxcvbn` algorithm.

## 🚀 Quick Start with Docker

You don't need to install Node.js or any npm packages locally. The application is fully containerized.

### Prerequisites
* Docker Desktop installed and running.

### Running the App
1. Clone this repository.
2. Open your terminal in the project root folder.
3. Run the following command:

docker-compose up --build

4. Open your browser and navigate to:
    http://localhost:5173
